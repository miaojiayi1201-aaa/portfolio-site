# GitHub Pages 发布说明

## 文件结构
- index.html：首页
- project1.html ~ project6.html：作品详情页
- style.css：统一样式
- images/：所有作品图片

## 发布步骤
1. 在 GitHub 新建一个公开仓库。
2. 把本文件夹内所有文件上传到仓库根目录。
3. 打开仓库 `Settings` → `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. 选择 `main` 分支和 `/ (root)`。
6. 保存后等待几分钟，即可获得网站链接。

## 链接格式
通常为：
`https://你的用户名.github.io/仓库名/`
